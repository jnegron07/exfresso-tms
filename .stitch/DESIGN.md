# Exfresso TMS Design System

## Visual Identity
- **Primary Color**: Navy (#1B2A4A) - Used for sidebar, primary headings, and deep backgrounds.
- **Secondary Color**: Teal (#0D9488) - Used for actions, highlights, and secondary UI elements.
- **Accent Colors**: 
  - Amber (#F59E0B) - Status: Pending/Warning.
  - Emerald (#10B981) - Status: Success/Completed.
  - Rose (#F43F5E) - Status: Error/Alert.

## Typography
- **Primary Font**: Plus Jakarta Sans (Web) or Inter.
- **Headings**: Bold, sharp.
- **Body**: Readable, optimized for data-heavy views.

## Component Styling
- **Corners**: Medium roundness (0.625rem / 10px).
- **Surface**: Glassmorphism effects with subtle blurs and borders.
- **Elevation**: Soft shadows, depth via layered backgrounds.

## Tone & Vibe
- **Professional**: Clean lines, no clutter.
- **Intelligent**: Visual indicators of AI processing (e.g., animated gradients, subtle glows).
- **Dynamic**: Micro-animations on hover and state transitions.
